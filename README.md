# The Stream Church Website

Public website for Mission Baptist Church: The Stream, serving the Spanish-speaking community in Warren, New Jersey.

## Live Links

- Live website: [thestreamchurch.org](https://thestreamchurch.org)
- GitHub Pages preview: [titosoriano.github.io/thestreamchurchweb](https://titosoriano.github.io/thestreamchurchweb/)
- Repository: [github.com/titosoriano/thestreamchurchweb](https://github.com/titosoriano/thestreamchurchweb)

## Overview

This project is a multi-page church website built with HTML, CSS, and JavaScript. It includes ministry pages, church information, media content, downloadable resources, and event-focused content designed for both desktop and mobile visitors.

## Highlights

- Responsive multi-page website
- Custom navigation and shared components
- Image, video, audio, and PDF content
- Spanish-language church and ministry content
- GitHub Pages preview deployment
- Production website available on a custom domain

## Built With

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Pages Included

- Home
- Mission and Vision
- What to Expect
- Our Pastors
- Offerings
- Missions and Outreach
- Grace Foundation
- Confession of Faith
- Baptist Catechism
- Tribute page for Pastor Santos Sandoval

## Purpose

The goal of this website is to provide an accessible digital home for The Stream Church, helping visitors learn about the church, explore ministries, access biblical resources, and connect with the congregation online.
