We believe in the power of community. Whether you are a seasoned developer or just learning HTML/CSS, your help is valuable!

### Ways to Help:
1.  **Code:** Help fix responsive issues or add new HTML sections.
2.  **Design:** Suggest color palettes, font pairings, or layout improvements.
3.  **Content:** Write copy for the "About" page or proofread existing text.
4.  **QA:** Test the site on your phone or different browsers and report bugs.

### Getting Started

Since this project uses vanilla HTML and CSS, there is no installation required!

1.  **Fork** and **Clone** the repository:
    ```bash
    git clone https://github.com/titosoriano/thestreamchurchweb.git
    ```
2.  **Open the project:**
    Navigate to the project folder.
3.  **Run the site:**
    Simply open the `index.html` file in your favorite web browser.
    *Alternatively, if you are using VS Code, use the "Live Server" extension for hot reloading.*
4.  **Make Changes:**
    Edit the HTML or CSS files.
5.  **Submit a Pull Request:**
    Push your changes and open a PR for review.
